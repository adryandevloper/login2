🔐 Tela de Login Básica
Este é um projeto inicial de uma interface de login simples, desenvolvida para praticar a manipulação do DOM com JavaScript e estilização com CSS.

🛠️ Tecnologias
HTML5: Estruturação da página.

CSS3: Estilização (arquivo index.css).

JavaScript: Lógica de captura de dados e validação.

📂 Estrutura de Arquivos
index.html: Contém os campos de entrada (Nome e Senha) e o botão de envio.

index.css: Arquivo responsável pelo layout e cores.

script.js: Responsável por processar o clique no botão e validar as informações.

🚀 Como usar
Certifique-se de que os três arquivos (index.html, index.css e script.js) estão na mesma pasta.

Abra o arquivo index.html em seu navegador.

Digite o nome, a senha e clique em Enviar.

📝 Funcionalidades (JavaScript)
Captura dos valores digitados nos campos.

Exibição de alertas para interação com o usuário.

Possibilidade de expansão para sistemas de autenticação reais.
